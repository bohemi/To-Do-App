# React + Vite To-Do App

A simple React + Vite to-do manager with category filters and task completion state.

Live Demo: https://bohemi.github.io/To-Do-App/

## Features
- Add tasks
- Categorize tasks
- Mark as complete
- Remove tasks
- Category filtering
- Responsive UI with Tailwind

## Tech Used
- React
- Vite
- TailwindCSS
